# 贡献指南

欢迎贡献！请遵循以下规范：

## 如何参与
1. Fork 本仓库
2. 创建特性分支: `git checkout -b feature/your-idea`
3. 提交更改: `git commit -m "描述你的修改"`
4. 推送分支: `git push origin feature/your-idea`
5. 提交 Pull Request

## 代码规范
- 遵循现有代码风格
- 提交信息使用英文或中文（保持统一）
- 新增功能需包含单元测试

## 问题报告
- 在 Issues 中描述问题复现步骤
- 提供环境信息（如操作系统、版本号）
